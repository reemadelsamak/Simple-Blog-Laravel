# Simple-Blog-Laravel
Simple App to apply CRUD operations and ability to login via Gmail / Github

## Project setup
- Create new mysql database called "laravel" and run :
```
npm install
```
```
php artisan migrate:fresh
```
## Run the Project
```
php artisan serve
```
